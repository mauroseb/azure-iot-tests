# Azure platform IoT Tests

## Description

The purpose of this repo is to host a number of files to test the functionality of Azure IoT-related products like:

- IoT Hub
- IoT Central
- IoT Edge
- Event Hubs

## Structure

Each folder will contian a specific test with its associated files related to the services above as well as the instructions to execute it.

## Source

Most of the examples are related to the official Azure documentation plus some additions.
